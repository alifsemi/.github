# Alif Semiconductor Software

Alif Semiconductor is the industry-leading supplier of the next-generation Ensemble family of 32-bit AI/ML (Artificial Intelligence & Machine Learning) microcontrollers and fusion processors. Alif Ensemble is the only choice for power efficient microcontrollers that can handle heavy AI and ML workloads, without draining a deviceâ€™s battery life.

The software here is delivered by [Alif Semiconductor](https://alifsemi.com/) as part of the Software Development Kit (SDK). The SDK includes supported Boards/Kits, Bare Metal Drivers, RTOS, Applications and Guides. Listed below are the contents of the SDK with relevant links and references to other documentation.
